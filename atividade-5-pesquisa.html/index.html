Elemento html

o Elemento < b > é usado para deixar o texto em negrito, sem indicar importância extra ou ênfase. Ele apenas muda a aparência do texto, o destacando. Esse recurso é útil para chamar atenção para certas palavras ou frases, mas sem alterar o significado

Exemplo:

    
Este texto tem uma palavra em  negrito

Resultado: Este texto tem uma palavra em negrito.
